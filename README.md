# Landing Page - The Odin Project

This is a solution to the Landing Page project from The Odin Project's Foundations course. This project focuses on implementing a layout using CSS Flexbox.
The Challenge

* The goal was to create a custom landing page based on a provided design mockup, ensuring:

* Proper use of Flexbox for alignment.

* Correct image scaling and positioning.

* Responsive design principles (optional but recommended).

* Clean, semantic HTML.
